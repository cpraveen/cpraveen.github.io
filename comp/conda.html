Python using Anaconda

I prefer to install Miniconda and then install whatever packages I need. 

conda update conda
conda install nomkl  # use openblas instead of mkl
conda update --all
conda install numpy scipy sympy matplotlib ipython jupyter tensorflow-eigen
conda install pylint autopep8 # vscode wants these

I set PATH to point to anaconda directory 

export PATH=/path/to/anaconda/bin:$PATH

See installed packages 

conda list

Periodically, delete older versions of packages 

conda clean --all

Install Mayavi

conda install mayavi
pip install PyQt5

To upgrade pip installed packages 

pip install --upgrade PyQt5 PyQt5-sip
